# Mobile Apps Directory

Since the `flutter` command was not found on your system, these placeholder directories were created for the customer and rider mobile applications.

## How to Initialize the Flutter Projects

Once you have installed the Flutter SDK, navigate to this directory in your terminal and run the following commands to initialize the projects:

### 1. Customer App
```bash
cd mobile-apps/customer_app
flutter create --org com.max.delivery.customer ./
```

### 2. Rider App
```bash
cd mobile-apps/rider_app
flutter create --org com.max.delivery.rider ./
```
